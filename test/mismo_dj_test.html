üí° Quick Start:
1. Start the Mismo DJ server: ./build/MismoDJ_artefacts/Debug/MismoDJ
2. Click "Connect" to establish WebSocket connection
3. Select your audio output device from the dropdown (auto-loads on connect)
4. Load tracks by file path OR by track ID (Phase 3.15)
5. Start mixing!

üÜî Load by Track ID (Phase 3.15): Enter a track ID and click "Load by ID". The audio server will request track metadata (BPM, key, beats) from the app server and load the track. This requires the app server to be connected!

üìÇ Load by Path: Enter file paths for tracks (absolute paths only - no ~/) and click "Load (Path)".

üîä Audio Device: You can switch output devices on-the-fly! The audio will briefly pause (~100-500ms) during the switch.

üéß PFL (Pre-Fader Listen): Click the "PFL" button on any deck to preview it in your headphones before bringing it into the mix. Use the Cue/Mix control to blend between PFL audio and the master mix. Perfect for beatmatching!

üìù Note: All commands are queued and executed by the audio engine. Watch the server console for execution results.
